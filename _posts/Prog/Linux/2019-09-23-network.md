---

1. Network interface 확인

        $ ifconfig -a

<br>
2. NetPlan을 통한 static IP 설정

        $ vi /etc/netplan/50-cloud-init.yaml

<br>
3. `address`, `gateway4`, `nameservers/addresses`(dns 서버) 정보를 변경
<br>
4. 설정 적용

        $ sudo netplan apply
