### 1. 추가된 disk volume 확인

    $ sudo fdisk -l
    $ lsblk

<br>
### 2. Filesystem 생성

    $ sudo mkfs.ext4 /dev/vdb
    // ext4 대신 ntfs 사용가능 (Read: ext4 < ntfs, Write: ext4 >> ntfs)
    // vdb 이름이 다를 수 있다.

<br>
### 3. Mount 받을 directory 생성

    $ sudo mkdir /workspace

<br>
### 4. Mount 하기

    $ sudo mount /dev/vdb /workspace
    $ df -h   // 확인
