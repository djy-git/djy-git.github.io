`*.bat` batch file을 통해 `bash`의 명령어들을 수행시킬 수 있습니다.  

### 1. For loop
C 문법과 유사한 형태로 사용할 수 있습니다.

```bash
for((i = 0 ; i <= 10 ; ++i))
do
  for((j = 0 ; j <= 10 ; ++j))
  do
    python main.py --i $i --j $j
  done
done
```


### 2. If statement

```bash
if [ "$IP" == "192.168.0.1" ]
then
  echo "Firetruck"
elif [ "$IP" == "192.168.0.2" ]
then
  echo "Mother"
fi
```

`[  ]` 조건문이 대괄호와 닿지 않아야 합니다.
