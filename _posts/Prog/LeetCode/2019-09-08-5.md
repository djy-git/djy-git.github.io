## Algorithm
{% highlight pseudo-code linenos %}
Input: s

n ← length(s)
max_len ← 0
rst ← ""
for start ← 0 to n - 1
  find all positions whose value is equal to that of start
  for pos in all positions
    ss = s[start:pos + 1]
    if ss is palindromic
      if length(ss) > max_len
        max_len ← length(ss)
        rst ← ss
      else
        break
    if start = (n - max_len) - 1
      return rst

{% endhighlight %}
