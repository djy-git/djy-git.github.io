`set` type의 operation으로 `union()`, `intersection()`, `difference()`, `symmetric_difference()`가 있다.


```python
a = set(['a', 'ab'])
b = set(['b', 'ab'])
```


```python
a.union(b)
```




    {'a', 'ab', 'b'}




```python
a.intersection(b)
```




    {'ab'}




```python
a.difference(b)
```




    {'a'}




```python
b.difference(a)
```




    {'b'}




```python
a.symmetric_difference(b)
```




    {'a', 'b'}
