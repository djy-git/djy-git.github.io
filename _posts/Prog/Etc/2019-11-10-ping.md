# Windows

### 1. PC, 공유기의 내부 IP 주소 확인

    > ipconfig

    Windows IP 구성

    이더넷 어댑터 로컬 영역 연결:
      ...
      IPv4 주소: {PC의 IP 주소}
      ...
      기본 게이트웨이: {공유기의 IP 주소}
만약 공유기를 경유하지 않고 바로 모뎀에 PC를 연결한 경우 PC의 IP 주소가 내부 IP 주소가 아닌 외부 IP 주소로 나타나게 됩니다.
<br>

### 2. Ping test (PC ↔ 공유기)

    > ping -t {공유기의 IP 주소}
<br>
